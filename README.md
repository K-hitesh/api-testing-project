# 🧪 API Testing Automation Project

This project automates testing of REST APIs using:

- ✅ Python + Pytest
- ✅ `requests` for making HTTP calls
- ✅ `pytest-html` for generating test reports
- ✅ Modular structure for scalability and reusability
